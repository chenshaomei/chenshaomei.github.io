vue指令

发表于 2017-10-23 | 分类于 vue
  • v-model ：在表单控件或者组件上创建双向绑定；
  • v-bind ：绑定需响应地更新元素的属性，通常是HTML元素的属性，缩写是： :href=”xx”
  • v-on ：绑定事件，缩写是：@click=”showModel”
  • v-html ：绑定标签内显示的内容，html标签在渲染的时候被解析了

  • v-text：绑定标签内显示的内容，html会被当做源码显示出来

    提示： v-text 和v-html的区别：v-text：内容里面的html不会被解析，会当成源码渲染，v-html：内容里面的html会被解析渲染。

  • v-show：根据表达式之真假值，切换元素的 display CSS 属性，值为真显示，假隐藏

  • v-if：根据表达式的值的真假条件渲染元素。在切换时元素及它的数据绑定 / 组件被销毁并重建。如果元素是
  • v-else ：条件语句
  • v-else-if：条件语句，前一兄弟元素必须有 v-if 或 v-else-if，可以链式调用

提示： v-show 和 v-if 的区别：v-show：只是控制元素的display切换，v-if：是切换元素在页面上渲染显示。

  • v-for：循环语句，遍历渲染。
  • v-cloak：这个指令保持在元素上直到关联实例结束编译。和 CSS 规则如 [v-cloak] { display: none }一起用时，这个指令可以隐藏未编译的 Mustache 标签，直到实例准备完毕。

  • v-once ：元素和组件只渲染一次。随后的重新渲染，元素/组件及其所有的子节点将被视为静态内容并跳过。这可以用于优化更新性能。

  • v-pre：跳过这个元素和它的子元素的编译过程。可以用来显示原始 Mustache 标签。跳过大量没有指令的节点会加快编译。
